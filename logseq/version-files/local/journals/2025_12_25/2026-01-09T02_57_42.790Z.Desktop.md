- 16:46 应该用 Flex 还是 Grid #reading
	- | **场景** | **推荐方案** | **原因** |
	  | ---- | ---- | ---- |
	  | **一维布局** (如导航栏、按钮组) | **Flex** | 只需要控制单向排列，Flex 更简单直观。 |
	  | **整体页面架构** (Header/Main/Footer) | **Grid** | 二维结构，Grid 的区域定义（Areas）非常清晰。 |
	  | **卡片网格列表** (Gallery, Products) | **Grid** | `gap` 和 `auto-fit` 能完美处理间距和响应式，Flex 需要负 margin hack。 |
	  | **内容垂直居中** | **Flex/Grid 皆可** | 简单的居中 Flex 够用了；如果不想加额外的 wrapper，Grid 也行。 |
	  | **不定高度动画** | **Grid** | Grid 独门绝技。 |
