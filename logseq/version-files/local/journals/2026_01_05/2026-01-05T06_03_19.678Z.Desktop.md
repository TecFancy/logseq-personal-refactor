	- 12:54 [[IDB]] 原生方法实现单字段精确查询 #reading
		- 原生方法实现的单字段精确查询中，需要理解“主键”和“索引”的区别。
		- 要实现该功能需要先[初始化 DB](((6956acae-2332-4bd6-9457-5ea1dc9dedd9)))
		- ```ts
		  const getTodosByStatus = (db: IDBDatabase, status: 'done' | 'pending') => {
		    return new Promise((resolve, reject) => {
		      try {
		        // 1. 开启事务
		        const tx = db.transaction(['todos'], 'readonly');
		        
		        // 2. 获取存储仓库
		        const store = tx.objectStore('todos');
		        
		        // 3. KEY: 获取索引
		        // 确保初始化 DB 时创建了这个索引
		        const idx = store.index('by_status');
		        
		        // 4. 执行查询
		        // 注意：getAll 查所有数据，大数据查询会卡死
		        const request = idx.getAll(status);
		        
		        // 5. 监听查询成功事件
		        request.onsuccess = () => {
		          console.log(`Found ${request.result.length} items.`);
		          resolve(request.result);
		        };
		        
		        // 6. 监听查询失败事件
		        request.onerror = () => {
		          reject(request.error);
		        };
		      } catch (error) {
		        reject(error);
		      }
		    });
		  };
		  ```
